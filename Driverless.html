Formula Student Driverless (FSD) challenges teams to build cars that navigate a cone-defined track completely autonomously — without any human input. From raw sensor data to tire contact patch in milliseconds.

Our system combines LiDAR, stereo cameras, and IMU data with custom perception and planning algorithms developed entirely in-house by students.

Technical Stack

[ SENSOR_01 ]
Primary LiDAR
64 ch
360° rotating LiDAR. Full point cloud at 20Hz. Cone detection up to 30m.

Stereo Camera
120 fps
120° field of view. Color-based cone classification. Fused with LiDAR detections.

IMU + GNSS

[ ALGO_01 ]
Path Planning
MPC ctrl
Model Predictive Control. Optimises lap time while respecting vehicle dynamics limits.
[ ALGO_02 ]
Mapping & SLAM
EKF SLAM
Extended Kalman Filter SLAM. Simultaneous mapping of cones while localising the car.

How It Works

👁
Perception
LiDAR + camera fusion. Cone detection & colour classification.
🗺
SLAM
Build a map of cones while tracking the car's position in real time.
🧠
Planning
Generate an optimal path through the track. Update every 50ms.
⚡
Control
MPC sends steering + throttle commands to the car's actuators.
🏎
Actuate
Drive-by-wire system executes commands. Lap complete. Repeat.
